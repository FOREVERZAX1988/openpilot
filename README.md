## üêé What is hoofpilot?
**hoofpilot** is a fork of sunnypilot, an open-source driver assistance system. **hoofpilot** offers an intuitive, simple-to-use experience for driver assistance, supporting over 300 different car makes and models.

## ‚ùî Supported devices

**hoofpilot** is designed around Konik.ai's A1 and A1M. **hoofpilot** also supports the comma 3x and the comma three devkit.

**Additional notes:**
* The comma four is not supported at the moment. However, support for the comma four will come in the near future.

## üß∞ Installation

The easiest way to install **hoofpilot** is by entering this URL on the installation screen:

```
https://hp.konik.ai/release
```

Additionally, you can install any custom branch of **hoofpilot** by using the following URL and your custom branch's name:

```
https://hp.konik.ai/branch-name
```

## ‚ö†Ô∏è Safety

**hoofpilot** is an Adaptive Cruise Control (ACC) and Automated Lane Centering (ALC) system. Like other ACC and ALC systems, **hoofpilot** is a failsafe passive system and it requires the driver to be alert and to pay attention at all times. **hoofpilot** is designed to comply with multiple established safety standards.

Read `docs/SAFETY.md` for more information.
